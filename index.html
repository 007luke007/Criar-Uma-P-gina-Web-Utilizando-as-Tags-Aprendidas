A História da Web: De 1989 até Hoje

  1. O Nascimento (1989 - 1991)

  3. Web 2.0: A Era Social e Interativa (~2004 - ~2014)

1. O Nascimento (1989 - 1991)

Voltar

A Web não é a mesma coisa que a Internet (a internet, a estrutura de cabos e redes, já existia). A Web foi inventada para dar sentido a essa rede.

O Pai da Web: O físico britânico Tim Berners-Lee, trabalhando no CERN (Suíça), queria uma maneira fácil de cientistas compartilharem documentos entre computadores diferentes.

A Trindade: Em 1990, ele criou as três tecnologias que você estuda até hoje:

HTML: A linguagem de marcação (o conteúdo).

URL: O endereço único de cada página.

HTTP: O protocolo de transferência (como o navegador "conversa" com o servidor).

O Primeiro Site: Foi ao ar em 1991. Era apenas uma página de texto preto com links azuis explicando o que era a Web.

2. Web 1.0: A Era Estática (1991 - ~2004)

Voltar

Fase da "Web somente leitura" (Read-Only).

Como era: Os sites pareciam jornais digitais. O dono do site publicava o conteúdo e você apenas lia. Não havia "curtir", "comentar" ou "fazer login".

Era rudimentar. Muito texto, cores neon, GIFs piscando e tabelas usadas para layout (o que hoje é considerado má prática). A Guerra dos Navegadores: O Netscape Navigator dominava, mas a Microsoft lançou o Internet Explorer. Eles brigavam criando tags proprietárias que só funcionavam no navegador deles. A Solução (W3C): Tim Berners-Lee fundou o W3C (World Wide Web Consortium) em 1994 para padronizar o HTML, garantindo que um site funcionasse igual em qualquer navegador (o que nos permite estudar um padrão único hoje).

3. Web 2.0: A Era Social e Interativa (~2004 - ~2014)

Voltar

Web 2.0 foi a revolução do "Leitura e Escrita" (Read-Write).

Usuário no Comando: As pessoas deixaram de ser apenas consumidoras para serem criadoras. Surgiram Blogs, Orkut, Facebook, YouTube e Wikipedia.

Tecnologia: O JavaScript e o CSS ficaram poderosos. As páginas pararam de precisar "recarregar" a cada clique (graças ao AJAX). O termo "Nuvem": Começamos a guardar nossos arquivos na internet e não mais apenas no HD do computador.

4. A Web Moderna e o HTML5 (2014 - Hoje)

Voltar

É a era em que vivemos agora, frequentemente chamada de Web Semântica ou Web 3.0 (embora o termo 3.0 seja debatido).

Mobile First: O acesso pelo celular superou o desktop. Os sites precisaram se tornar responsivos (se adaptar a qualquer tela).

HTML5: Lançado oficialmente em 2014, trouxe padronização para vídeos e áudio (matando o antigo Adobe Flash) e tornou o código mais "inteligente" (semântico).

Web Apps: Hoje, ferramentas como Google Docs, Canva ou Figma rodam no navegador com a mesma potência de programas instalados.
